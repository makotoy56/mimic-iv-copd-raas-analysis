# SAS Analysis Workflow

This directory contains an independent SAS workflow used to reproduce selected statistical analyses from the Python-based analysis.

## Structure

- inputs/: exported analysis-ready datasets used by SAS
- programs/: SAS programs
- outputs/: tables, listings, and model results generated by SAS
- logs/: SAS log files

## Purpose

The SAS workflow is used to:
1. import exported cohort datasets
2. generate baseline tables
3. reproduce the primary Cox proportional hazards model
4. export logistic regression parameters only as a secondary Python-vs-SAS binary-outcome validation artifact
5. compare key results with Python outputs
